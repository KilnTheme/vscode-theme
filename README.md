# Kiln - VS Code Theme

Warm, earthy terracotta color theme in **Dark** and **Light** variants, matching the Rether design system.

## Install (local)

1. Copy this whole `vscode-theme` folder into your VS Code extensions directory, renamed to something like `kiln-vscode-theme`:
    - **macOS / Linux:** `~/.vscode/extensions/kiln-vscode-theme/`
    - **Windows:** `%USERPROFILE%\.vscode\extensions\kiln-vscode-theme\`
2. Fully restart VS Code.
3. Open the Command Palette (`Cmd/Ctrl+Shift+P`) → **Preferences: Color Theme** → pick **Kiln Dark** or **Kiln Light**.
